# Day 08 - Alternate Numbers in Single Line 🔢

## Problem Statement

Take two integer inputs (`start` and `end`) and print every alternate number between them in a single line separated by a symbol.

### Example

Input:
start = 1
end = 10

Output:
1_3_5_7_9
