# Day 4 - Agent Badge Number 🪪

## Problem Statement

An agent has the following properties:

- agent_letter = "B"
- code_number = "5"
- security_level = "2"

The badge number is calculated by adding the ASCII values of all three properties.

---

## Concepts Used

- Variables
- Strings
- ASCII Values
- `ord()` Function
- Arithmetic Operators

---

## Formula

Badge Number = ord(agent_letter) + ord(code_number) + ord(security_level)
