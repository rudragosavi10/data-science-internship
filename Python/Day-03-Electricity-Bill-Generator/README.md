# Day 3 - Electricity Bill Generator ⚡

## Problem Statement

Generate an electricity bill using the following slabs:

- First 100 units → Free
- Next 100 units → ₹5 per unit
- Next 100 units → ₹7 per unit
- Above 300 units → ₹10 per unit

---

## Concepts Used

- Variables
- Input and Output
- Conditional Statements (`if`, `elif`, `else`)
- Arithmetic Operators
- Slab Based Calculations
- Logic Building

---

## Edge Cases Handled

- Negative units are treated as invalid input.
